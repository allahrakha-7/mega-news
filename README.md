# MegaNews Analytics

An interface aggregating multi-source insights leveraging NewsAPI endpoints. Built with a focus on strict latency tolerance, clean code and comprehensive error handling. 

## Installation & Setup 

1. Clone the repository and run `npm install` to install dependencies. 
2. Create a `.env.local` file in the root directory and add your NewsAPI key:
```bash
NEXT_PUBLIC_NEWS_API_KEY=your_news_api_key_here
```
3. Run `npm run dev` to start the development server.
